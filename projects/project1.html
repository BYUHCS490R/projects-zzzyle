1. Internet

  • A network that provides various information and communication from one device to another. It is the physical infrastructure: wires, cables, and serves, that allow each other to communicate.

2. Web

  • In computing, the "web" refers to the World Wide Web, a system of documents and resources that can be accessed through the Internet.

3. How are the Internet and Web different?

  • The Internet and Web different in a way that the Internet is a global network infrastructure of interconnected devices. While World Wide Web (www) is a service that uses this infrastructure to access information. Examples of the web is sites such as www.churchofjesuschrist.org.

4. ASCII and Unicode

  • ASCII: or the American Standard Code for Information Interchange is used before as a standard for encoding characters. It uses a small set of numbers (0-127) to represent English letters, numbers, and basic symbols. Each character has a corresponding symbol in decimal and hex.
  • Unicode: is a more modern and comprehensive standard that can represent almost every character from every written language in the world, including emojis and special symbols and characters like Chinese, Japanese, etc. It is like a massive multilingual dictionary.

5. What is UTF-8

  • UTF stands for Unicode Transformation Format. It is the most common way to encode or save unicode characters into format computers would understand. It uses the variable number of bytes 1 to 4.

6. URL

  • is an abbreviation of Uniform Resource Locator. This is the complete address of a web page. You type this in the search bar for the browser to find a specific page. For example: https://my.byuh.edu/dashboard/

7. Domain Name

  • is the part of a URL where it is easy to remember. For example: mybyuh.edu. The network address that identifies it as belonging to a particular domain.

8. HTTP

  • HTTP stands for HyperText Transfer Protocol and is the system for the internet. When you type a web address into your browser, HTTP will then send that request to the websites server and brings the content whether it maybe text, image, video back to your screen.

9. HTTPS

  • HTTPS stands for Hypertext Transfer Protocol Secure the secure version of the standard HTTP. It encrypts communication between your browser and a website and protect your information.

10. IP Address

  • An IP address which stands for Internet Protocol address is a unique label consisting of numbers that identifies a device on the Internet, like 172.217.10.14. This is the "real address" that computers or even printers use to find each other.

11. DNS

  • DNS or the Domain Name System is the "Internet's phone book". When typing a domain name into the the browser, DNS translates that name into its corresponding IP Address so that the device knows where to go.

12. W3C Recommendations

  • W3C stands for World Wide Web Consortium. This is an international community that develops standards and rules for the Web to ensure its growth and improvement. W3C Recommendations serves as a guideline for web technologies.

13. XML, XHTML, HTML, HTML5

  • XML (Extensible Markup Language): is a more flexible markup language used to store and transport data. Unlike HTML, which has already given tags, XML will let you create your own tags to describe content or data.
  • XHTML (Extensible HyperText Markup Language): is a stricter, cleaner version of HTML that follows XML's rules. This just means that every tag myst be properly closed.
  • HTML (HyperText Markup Language): is the basic language to create web pages. It uses tags like p for paragraph, or h for head to structure content and tell a browset how to display it.
  • HTML5: HTML5 is the latest version of HTML. It is more flexible and updated. It introduces new features, making it easier for developers to use this.
