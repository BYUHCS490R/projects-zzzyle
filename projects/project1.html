Internet

  • a global computer network providing a variety of information and communication facilities, consisting of interconnected networks using standardized communication protocols.

Web

  • computing, the "web" refers to the World Wide Web, a vast system of interlinked documents and resources accessed through the Internet

How are the Internet and Web different?

  • Internet is the global network infrastructure of interconnected computers, while the World Wide Web (WWW) is a service that uses this infrastructure to access linked information

ASCII and Unicode

  • ASCII
  • Unicode

What is UTF-8

  • UTF-8 is the most common character encoding system for the internet, representing every character in the Unicode Standard using a variable number of bytes (1 to 4).

URL

  • the address of a web page.

Domain Name

  • the part of a network address that identifies it as belonging to a particular domain.

HTTP

  • HTTP stands for HyperText Transfer Protocol and is the foundational protocol for data exchange on the World Wide Web, allowing web clients (like your browser) to request and receive resources such as web pages, images, and videos from web servers.

HTTPS

  • HTTPS is the secure version of the standard Hypertext Transfer Protocol (HTTP) that encrypts communication between your web browser and a website.

IP Address

  • An IP address (Internet Protocol address) is a unique numerical label assigned to every device on a computer network that uses the Internet Protocol for communication

DNS

  • DNS (Domain Name System) is the "phone book" of the internet, translating human-friendly domain names (like www.google.com) into machine-readable IP addresses (like 172.217.160.142) that computers use to locate each other and deliver web content.

W3C Recommendations

  • A W3C Recommendation is a formal, endorsed web standard or guideline for web technologies, such as HTML and CSS, developed by the World Wide Web Consortium (W3C) after a rigorous consensus-building process involving W3C members and the community
