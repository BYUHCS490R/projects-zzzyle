1. Explain the difference between inline, embedded, and external CSS. Inline - is the CSS is within the paragraph, it formats the each paragraph instead of doing it as a whole. Embedded - is having the CSS withint the HTML before the header. External - is having a separate document for the CSS alone and linking the HTML page to it.

2. Explain why you should only use external CSS. You can link the CSS by typing link:css. Using External CSS will help companies be efficient working with Graphic Designers in changing the look of the page. It will help save time instead of changing CSS in every page. It will help with uniformity with the websites as well.

3.What is a selector used for? Selector is at the CSS page used to select and target specific HTML elements on the web page to apply the styling format on that.
