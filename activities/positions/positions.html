Engagement Questions

  1. What is normal flow in HTML? Normal flow in HTML is the default way the browser arranges elements on a webpage when no CSS positioning is given.
  2. What does float do? Why do you need to clear the float? It is used to position an element to the left or right of its containing element, allowing other content, such as text, to wrap around it.
  3. What do absolute, fixed, and relative do? Absolute is the positioning places an element relative to its nearest positioned ancestor. Fixed positioning places it relative to the browser window, and relative positioning places it relative to its own normal position in the document flow.
  4. How can you make nice navigation buttons for mobile devices? Use text decoration, background color, padding, and display for nav this way it would be easier to click when users are on mobile.
