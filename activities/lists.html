5. What is the difference between OL and UL Tags?

UL (Unordered Lists) tags that show up as bullet points, where the order doesn't matter. While OL (Ordered Lists) tags can be customized lists with numbers (1,2,3), letters (a,b,c), or even roman numerals (i,ii,iii). This is done by adding an attributes after OL and selecting the desired unit for the list. This usually use when the order matters like steps or ranking.

6. What is a DL?

DL (Description List) tag, when you need to describe something and give information. Using <dl> and adding Title <dt> and the description <dd>
